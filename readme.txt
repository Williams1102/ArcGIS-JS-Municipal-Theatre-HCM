Chào thầy, để chạy được bài file html nhận được data nhóm 10, thầy vui lòng thực hiện các bước sau:
B1: Nhóm sử dụng library JSON-SERVER (https://www.npmjs.com/package/json-server) để chạy giả lập server. 
	Sử dụng NPM để tải global thư viện: 
	------------------------------
	| npm install -g json-server |
	------------------------------
B2: sau đó tại thư mục đồ án, thầy mở Terminal và chạy lệnh sau:
	------------------------------------------
	| json-server --watch benthanh-data.json  |
	------------------------------------------
B3: sau khi chạy lệnh json-server sẽ mở một server url: http://localhost:3000. 
Thầy mở file benthanh.html, và chầm bài thực hành
